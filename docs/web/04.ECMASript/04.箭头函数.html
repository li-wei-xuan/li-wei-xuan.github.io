# 箭头函数

使用「箭头」（=>）定义函数。

注意：箭头函数不会更改 this 指向，用来指定回调函数会非常合适
1
2
3
1
2
3

# 注意点

  1) 如果形参只有一个，则小括号可以省略
  2) 函数体如果只有一条语句，则花括号可以省略，函数的返回值为该条语句的执行结果
  3) 箭头函数 this 指向声明时所在作用域下 this 的值
  4) 箭头函数不能作为构造函数实例化
  5) 不能使用 arguments

# 代码

    // 正常写法
    let fn = (a,b) =>{
        return a+b
    }
    let a = fn(1,2)
    console.log(a) // 3

    // 如果形参只有一个，则小括号可以省略
    let fn = a =>{
        return a+a
    }
    let a = fn(1)
    console.log(a) // 2

    // 函数体如果只有一条语句，则花括号可以省略，函数的返回值为该条语句的执行结果
    let fn = a =>  a+a
    let a = fn(1)
    console.log(a) // 2

    // 箭头函数 this 指向声明时所在作用域下 this 的值
    let fn = () => {
     console.log(this) //window
    }

    const obj = {
        test() {
            setTimeout(function () {
                console.log(this) //window
            })

            setTimeout(() => {
                console.log(this) //test
            })
        }
    }
    obj.test()
